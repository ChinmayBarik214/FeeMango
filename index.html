Automated Fee Tracking

Our software keeps track of all fee data per student, which can be looked up or updated at any time. This will remove the need of all paperwork and simplify the payment process.

Secure Data Management

We make sure your data remains secure. All databases are password protected and safe from attackers.

Easy Setup

Just run the following commands on your mySQL command line client and then launch Fee Management.py:

  • • drop database school;
  • • create database school;
  • • use school;
  • • create table student(roll int(5) primary key, name varchar(20) not null, age int(2) not null, class int(3), city varchar(10));
  • • create table fee(roll int(5) references student(roll), FeeDeposit int(6) not null, month varchar(10));
